Pistou ou Pesto ?

Le mélange des deux traditions

"Pourquoi choisir ? J'emprunte la force du Sud-Est et la gourmandise de l'Italie."

Les Ingrédients de base

  • 🌿 3 beaux bouquets de basilic frais
  • 🧄 4 gousses d'ail dégermées
  • 🫒 Huile d'olive de Provence
  • 🧂 Gros sel de Camargue

La Préparation

1. La base commune : Dans le mortier, écrasez l'ail avec le gros sel pour obtenir une crème lisse.

2. Le choix du gourmand : Si vous choisissez la version italienne, ajoutez vos pignons ou amandes maintenant et pilez.

3. Le basilic : Ajoutez les feuilles de basilic. Travaillez avec le pilon en tournant contre les parois.

4. Le liant : Versez l'huile d'olive en filet. Si vous mettez du fromage, ajoutez-le à la toute fin.

Note de Marie : Les amandes apportent un croquant que j'adore !
